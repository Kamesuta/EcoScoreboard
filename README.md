# EcoScoreboard

Vaultの所持金をスコアボードに表示するプラグインです。

## 前提プラグイン
- Vault

## コンフィグ
```yaml
# 所持金を表示するスコアボードの名前
scoreboard: 'money'
```

## 検証済み環境

- Minecraft 1.20.4
- Paper 496
- XConomy 2.25.14
- Vault 1.7.3-b131
- Java 17 (Java HotSpot(TM) 64-Bit Server VM 17.0.1+12-LTS-39) Host: Windows 10 10.0 (amd64)
